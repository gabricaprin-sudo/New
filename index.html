دليل الاستخدام والحماية

🔐 تسجيل الدخول

كل خادم/خادمة لها PIN خاص (4 أرقام). PIN ده بيتسجل مع كل تعديل علشان نعرف مين عدل. لو نسيت الـ PIN، سجل كخادم جديد.

📝 إضافة مخدومة جديدة

من صفحة "الحضور اليومي" اضغط على زر "إضافة مخدومة" واكتب الاسم والسنة الدراسية. النظام يكشف التكرار تلقائياً.

⭐ تسجيل الحضور والتقييم

في بطاقة كل مخدومة، اضغط على زر "تسجيل" لفتح نافذة التسجيل. سجل الحضور لكل نشاط (دراسي، محفوظات، قبطي، الحان) وقيّم بـ 1-5 نجوم.

☁️ المزامنة والحماية

البيانات تُحفظ تلقائياً على Firebase. كل تعديل بيتسجل في السجل التاريخي مع اسم الخادم والوقت. البيانات محمية بـ Security Rules.

⚠️ التعارضات

لو حد تعدل البيانات وأنت شغال، النظام يظهر لك تنبيه بالتعارض واسم اللي عدّل.

💾 Backup

من صفحة التصدير، استخدم "إنشاء Backup" لحفظ نسخة محلية مشفرة. استخدم "استعادة Backup" لرفع النسخة تاني.
